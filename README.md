# Game Development Portfolio

This repository contains my game development projects, documenting my learning journey and technical evolution — from my first self-taught solo project to more complex team-built games.

Each folder represents a complete project with its own `README.md` detailing the technical challenges, solutions implemented, and the context behind its development.

## Key Learnings Summary

This journey provided a solid foundation across several areas of game development, spanning two distinct stages:

### 1. Fundamentals & Self-Direction (Escape the Purple House)

My first project was an exercise in initiative and independent problem-solving. Built in 2022 before LLMs became widely available, and before I started university, it cemented the following skills:

- **Game Logic:** Structuring a project end-to-end, managing game states (menu, gameplay, game over).
- **Basic AI:** Implementing a functional pursuit AI using state machines and pathfinding concepts to create a challenging enemy.
- **Self-Teaching:** The ability to research solutions through documentation and forums to independently overcome technical blockers.

### 2. Collaboration & Technical Leadership (Wildlife Tower Defense)

The second project was a significant jump in complexity and scale, placing me in a team environment as **Lead Developer**. The focus shifted to multiplayer online game development and team management:

- **Client-Server Architecture:** Hands-on understanding of state replication, security, and network communication (`RemoteEvents`/`RemoteFunctions`) in a multiplayer context.
- **Complex Systems:** Advanced features including `DataStoreService` for data persistence and **Linear Algebra** applied to tower and projectile mechanics.
- **Leadership & Code Management:** Managing and integrating contributions from nearly **10 developers**, conducting code reviews, and maintaining project stability throughout.
