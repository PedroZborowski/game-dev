## Sobre o Projeto

Este é o primeiro jogo que desenvolvi, marcando minha entrada no mundo do desenvolvimento de games. Criado em 2022, este projeto foi um desafio particular por ter sido desenvolvido em uma época onde o acesso a LLMs (como o ChatGPT) ainda não era popular. Isso tornou a busca por conhecimento e a solução de problemas uma tarefa mais complexa e investigativa, dependendo de documentações, fóruns e muita experimentação.

Além disso, foi um projeto totalmente autodidata, realizado antes do meu ingresso na faculdade, movido pela paixão por jogos e pela curiosidade em entender como eles funcionam.

## O Desafio: A Inteligência Artificial do Monstro

Apesar da simplicidade geral do jogo, meu principal foco e área de estudo foi a criação da IA para o monstro que persegue o jogador. Em vez de um comportamento simples e previsível, dediquei tempo para pesquisar e implementar diferentes algoritmos para criar uma perseguição que fosse desafiadora e imersiva. Apesar de possuir alguns bugs, foi um primeiro contato divertido.

O desenvolvimento da IA envolveu o trabalho com:

-   **Máquinas de Estado (State Machines):** Para controlar os diferentes comportamentos do monstro, como `PATRULHANDO`, `PERSEGUINDO` e `ATACANDO`.
-   **Detecção do Jogador:** Lógica para o monstro "sentir" a presença do jogador através de campo de visão.
-   **Algoritmos de Busca (Pathfinding):** Implementação de algoritmos para que o monstro pudesse navegar pelo cenário de forma inteligente, contornando obstáculos para alcançar o jogador.

## Tecnologias Utilizadas

-   **Motor/Engine:** Roblox Studio
-   **Linguagem:** Lua

## Principais Aprendizados

-   Desenvolvimento de sistemas de Inteligência Artificial para inimigos.
-   Gerenciamento de estados de jogo (menu, gameplay, game over).
-   Resolução de problemas de forma independente, utilizando documentações e fóruns como principal fonte de conhecimento.
-   Estruturação de um projeto de jogo do início ao fim, desde a concepção da ideia até um produto jogável.
